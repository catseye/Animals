Description

The classic 'expert system' demonstration game of Animals. This game stores a 'knowledge tree' about the animals it knows in a nested tuple structure. This is mainly to demonstrate how one can work with binary trees as Erlang terms. A more serious implementation would probably use a real database system, such as Mnesia.
